Como rodar localmente

Pré-requisitos: Docker Desktop instalado
Passo a passo:
1. Copie o arquivo de configuração: cp .env.example .env 2. Edite o .env com seu token WhatsApp e telefone: WHATSAPP_TOKEN=seu_token WHATSAPP_PHONE_ID=seu_phone_id MASTER_PHONE=5511999999999 3. Suba todos os serviços: docker compose up --build 4. Acesse o painel: http://localhost:3000 5. (Opcional) Ver logs do worker: docker compose logs -f worker
✅ Sem token WhatsApp, o sistema funciona normalmente — os envios são simulados no console do worker. Útil para testar o fluxo completo antes de configurar a API da Meta.

Endpoints da API

POST /api/patients — Cadastrar paciente GET /api/patients — Listar pacientes GET /api/patients/:id — Buscar paciente DELETE /api/patients/:id — Remover paciente POST /api/appointments — Criar agendamento (agenda jobs automaticamente) GET /api/appointments — Listar agendamentos GET /api/appointments/queue/status — Status da fila BullMQ DELETE /api/appointments/:id — Remover agendamento (cancela jobs)
